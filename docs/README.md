# BrightLearn.in

## Overview
BrightLearn.in is a web application designed to provide IT services and digital marketing solutions, modeled after Uptricks Services.

## specialized Structure
- **client/**: React.js frontend application (Vite + Tailwind CSS).
- **server/**: Node.js express backend server.
- **docs/**: Project documentation.

## Tech Stack
- **Frontend**: React, Tailwind CSS
- **Backend**: Node.js, Express
- **Database**: Supabase
